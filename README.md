# 宁波中心城区中考志愿模拟编排器

这是一个纯静态的中考志愿模拟页面，用于根据考生分数与位次，按“冲、稳、保”结构生成宁波中心城区普通高中志愿建议，并单独保留中职/综合高中备选位。

## 当前实现

- 输入页：填写考生姓名、分数、位次、邮箱（邮箱当前仅保留字段，未接入发送）
- 结果页：固定输出 8 个普通高中志愿
- 中职部分：固定保留 10 个备选位，当前统一标记为“待补充”
- 数据存储：结果写入浏览器 `localStorage`，避免通过 URL 传参导致页面打不开
- PDF 导出：通过浏览器打印功能导出，避免中文乱码问题

## 页面结构

- `index.html`：输入页
- `result.html`：结果页
- `data.js`：中心城区普高数据与中职占位数据
- `logic.js`：志愿编排逻辑
- `script.js`：表单提交、结果渲染、打印与分享逻辑
- `style.css`：页面与打印样式

## 推荐逻辑

### 普高志愿

系统从中心城区普高池中生成 8 个志愿，采用以下结构：

- 2 个冲刺位
- 3 个稳妥位
- 3 个保底位
- 如某一层级数量不足，再按位次接近度补位

核心依据是考生位次与学校 2025 年录取位次的差值。

### 中职/综合高中

当前只保留 10 个备选志愿位，不伪造学校与专业数据。待补充真实数据库后，再生成精确推荐。

## 数据说明

普通高中数据当前依据：

- 宁波市教育局 2024-2025 年城区统招分数线整理
- `references/2024-2025分数线对比表（按2025位次排序）.md`

当前学校池仅保留中心城区普高，不再混入非城区学校。

## 本地运行

这是一个静态页面项目，可直接用本地静态服务器打开。

例如：

```bash
python -m http.server 8124 --directory "/d/nmworkshop/D盘工作区/nb-zhongkao-recommender"
```

然后访问：

```text
http://127.0.0.1:8124/index.html
```

## PDF 导出说明

点击结果页“导出打印版 PDF”后，会调用浏览器打印。建议在浏览器打印面板中选择：

- 目标：另存为 PDF
- 纸张：A4
- 背景图形：按需开启

当前方案依赖浏览器原生中文字体渲染，已替代原先容易乱码的 jsPDF 中文导出方案。

## 当前限制

- 邮件发送尚未接入
- 分享链接只分享结果页地址，不包含本地结果数据
- 中职/综合高中数据库尚未补全
- 本工具仅供模拟参考，不能替代官方正式志愿填报
